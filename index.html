Tentang Aplikasi

Tobacco AI Grader adalah dasbor penilaian kualitas daun tembakau berbasis kecerdasan buatan. Aplikasi ini mengklasifikasikan daun ke dalam tiga grade — Premium (A), Menengah (B), dan Rendah (C) — secara langsung melalui kamera perangkat menggunakan model Teachable Machine dan TensorFlow.js yang berjalan sepenuhnya di browser tanpa server.

Pemindaian Real-time

Analisis daun tembakau langsung dari kamera tanpa perlu unggah gambar.

Model AI Terlatih

Klasifikasi grade menggunakan model Teachable Machine untuk akurasi tinggi.

Penilaian Visual

Menilai warna, tekstur, dan kondisi daun untuk menentukan grade kualitas.
